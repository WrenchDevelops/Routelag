# Troubleshooting

## RouteLag Engine Missing

Symptom: `RouteLag Engine is missing or damaged. Reinstall RouteLag.`

Fix: reinstall RouteLag. The installer must include the RouteLag Engine
resources.

## Previous Optimization Did Not Close Cleanly

Click Restore Internet. RouteLag treats missing or already-stopped services as
success, clears stale route session state, and flushes DNS.

## Normal Internet Broken

1. Click Restore Internet.
2. Confirm normal browsing works.
3. Only use Repair Windows Network if Restore Internet does not fix normal
   internet.

## Report ZIP

The ZIP includes RouteLag diagnostics, targeted route status, RouteLag Engine
status, app logs, and tester notes. Private keys are redacted.
