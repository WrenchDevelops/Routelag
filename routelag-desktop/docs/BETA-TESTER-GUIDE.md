# RouteLag South Africa Private Beta

RouteLag includes RouteLag Engine. Do not install a separate WireGuard app.

We are testing RouteLag for Fortnite Middle East routing from South Africa.
This is a private routing beta, not a public release and not a guaranteed ping
reduction.

## Test Flow

1. Install RouteLag.
2. Open RouteLag.
3. Click Restore Internet first.
4. Log in with your beta code.
5. Open Fortnite.
6. Set matchmaking region to Middle East.
7. Play 1 match with RouteLag OFF.
8. Screenshot ping and packet loss.
9. Test Johannesburg route.
10. Screenshot ping and packet loss.
11. End Optimization.
12. Test Frankfurt route.
13. Screenshot ping and packet loss.
14. End Optimization.
15. Test London route.
16. Screenshot ping and packet loss.
17. End Optimization.
18. Test Amsterdam or Paris route.
19. Screenshot ping and packet loss.
20. Export RouteLag report ZIP and send it back.

After every test, click End Optimization before switching routes.

If anything feels broken, click Restore Internet. Do not click Repair Windows
Network unless Restore Internet does not fix normal internet.

## Send Back

- Your city
- Your ISP
- Ethernet or Wi-Fi
- Ping and packet loss screenshots for RouteLag OFF and every route
- Best route
- Any internet issues
- RouteLag report ZIP
